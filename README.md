# HEALTH ARCHI スマート見積システム（木造在来工法専用）

木造在来工法の見積を、最初にマスタ（単価表）を登録しておけば、建物の規模を入力するだけで一瞬で概算見積が完成。明細の手直しもかんたん、見積の確定・変更見積・請求書まで一気通貫。

## 公開URL

- 本体：https://healtharchi.github.io/smart-quote-pro/
- 使い方ガイド：https://healtharchi.github.io/smart-quote-pro/guide.html

## 特徴

- 標準単価マスタ付き（自社単価に書き換えて永久使用）。在来軸組の積算ロジックに合わせた項目構成
- 延床坪数・階数・屋根形状などを入れるだけで、外周長・外壁・屋根などを自動計算して概算見積を即生成
- 明細の手直し：数量・単価の変更、項目の削除・追加が自由
- 端数のお値引き（1万円・10万円単位などで切り捨て＝「お値引き」として自動計上・固定値引き方式）
- **見積の確定 → 変更見積（変更前→変更後を視覚化）→ 請求書（分割対応）まで一気通貫**
- 3書式（シンプル／内訳付き／正式・押印欄付き）。印刷からPDF保存
- データはブラウザ内に保存（当社サーバーには一切送信されません）。設定からエクスポート／インポート可
